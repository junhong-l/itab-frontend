# 浏览器导航页扩展

一个简洁美观的浏览器新标签页扩展，采用 iOS 风格设计，支持文件夹管理、拖拽排序、隐私模式、多搜索引擎切换、时间显示（含农历）和个性化外观设置。

## 功能特性

### 🕐 时间显示
- 实时显示当前时间（时:分:秒）
- 显示公历日期和星期
- 显示农历日期和生肖年

### 🔍 搜索功能
- 内置多个搜索引擎（百度、Google、必应、搜狗、DuckDuckGo）
- 支持快速切换搜索引擎
- 支持自定义添加搜索引擎
- 支持管理（编辑/删除）搜索引擎
- 书签搜索功能，快速查找已保存的书签

### 文件夹管理（iOS 风格）
- 创建文件夹整理书签
- iOS 风格 3x3 预览图标网格
- 点击文件夹展开弹窗查看内容
- 拖拽文件夹调整排序
- 拖拽书签到文件夹进行整理

### 📌 书签管理
- 添加可点击的网站快捷方式
- 自动获取网站图标
- 支持编辑和删除书签
- 右键菜单快速操作
- **拖拽排序**：根目录和文件夹内都支持拖拽排序
- **钉到常用**：文件夹内的书签可"钉"到常用区域显示（右键 → 钉到常用）
- **多选模式**：批量选择和删除书签

### 🔒 隐私模式
- 双击日期区域的星期可切换隐私模式
- 隐私模式下只显示隐私文件夹和隐私书签
- 隐私模式完全隐藏，无任何视觉指示
- 隐私模式下创建的书签和文件夹自动标记为隐私

### 📚 书签导入
- 支持导入浏览器导出的 HTML 书签文件
- 预览书签并选择性导入
- 保留文件夹结构信息

### 💾 备份与恢复
- 导出所有数据为 JSON 文件
- 支持导入备份文件恢复数据
- 备份内容包括：
  - 所有文件夹（含隐私文件夹）
  - 所有书签（含钉住状态、隐私属性）
  - 搜索引擎配置
  - 外观设置

### 🎨 个性化设置
- **背景设置**
  - 渐变背景（可自定义颜色和角度）
  - 纯色背景
  - 自定义图片背景
- **外观设置**
  - 图标大小调节
  - 文件夹大小调节
  - 图标圆角弧度调节
  - 搜索框圆角弧度调节

## 安装方法

### Chrome / Edge / Brave 等 Chromium 浏览器

1. 下载或克隆本项目到本地
2. 打开浏览器扩展管理页面
   - Chrome: `chrome://extensions/`
   - Edge: `edge://extensions/`
3. 开启右上角的「开发者模式」
4. 点击「加载已解压的扩展程序」
5. 选择本项目文件夹

### Firefox

Firefox 需要将 `manifest.json` 中的 `manifest_version` 修改为 `2`，并做相应的 API 调整。

## 使用说明

### 添加书签
1. 点击右下角「+」按钮
2. 选择「添加书签」
3. 填写网站名称和网址
4. 选择所属文件夹（可选）
5. 点击保存

### 添加文件夹
1. 点击右下角「+」按钮
2. 选择「添加文件夹」
3. 填写文件夹名称
4. 点击保存

### 编辑/删除书签
- 右键点击书签图标，选择「编辑」或「删除」

### 钉到常用
- 右键点击**文件夹内**的书签
- 选择「钉到常用」
- 书签会在常用区域显示快捷入口（带图钉标记）
- 原书签仍保留在文件夹内
- 再次右键可「取消钉住」

### 拖拽排序
- **书签排序**：拖拽书签到其他书签的左侧/右侧
- **文件夹排序**：拖拽文件夹到其他文件夹的左侧/右侧
- **移动到文件夹**：拖拽书签到文件夹上
- **移出文件夹**：在文件夹弹窗内拖拽书签到弹窗外

### 隐私模式
1. 双击日期区域的「星期X」文字
2. 进入隐私模式，只显示隐私内容
3. 再次双击退出隐私模式
4. 隐私模式下创建的书签/文件夹自动为隐私

### 多选模式
1. 点击右下角「✓」按钮进入多选模式
2. 点击书签进行选择/取消选择
3. 可使用「全选」「取消全选」按钮
4. 点击「删除选中」批量删除
5. 点击「退出多选」结束

### 切换搜索引擎
- 点击搜索框左侧的搜索引擎名称
- 在下拉菜单中选择想要使用的搜索引擎

### 管理搜索引擎
1. 点击右下角设置按钮
2. 切换到「搜索引擎」选项卡
3. 可以添加、编辑或删除搜索引擎

### 导入书签
1. 首先从浏览器导出书签为 HTML 文件
2. 点击右下角设置按钮
3. 切换到「导入书签」选项卡
4. 选择导出的书签文件
5. 确认导入

### 备份与恢复
1. 点击右下角设置按钮
2. 切换到「导入书签」选项卡
3. **导出备份**：点击「导出备份」按钮，下载 JSON 备份文件
4. **导入备份**：点击「导入备份」按钮，选择之前导出的 JSON 文件

### 自定义外观
1. 点击右下角设置按钮
2. 在「背景设置」中选择背景类型并自定义
3. 在「外观设置」中调整图标大小、文件夹大小、圆角弧度

## 数据存储

所有设置和数据都保存在浏览器本地存储中：
- 扩展环境：使用 `chrome.storage.local`
- 普通网页：使用 `localStorage`

## 技术栈

- 原生 HTML5 / CSS3 / JavaScript (ES6+)
- Chrome Extension Manifest V3
- 农历算法（支持 1900-2100 年）
